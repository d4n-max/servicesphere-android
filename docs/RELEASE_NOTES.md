# Release Notes

## Version 1.0.0
- Removed a misleading hamburger icon from the top bar where bottom navigation is the active app navigation model.
- Added Settings actions for rating ServiceSphere on Google Play and reading in-app Help & Support FAQs.
- Added a guided walkthrough for new users and a Settings option to replay it.
- Improved launcher icon spacing and safe-zone rendering.
- Improved app icon assets so the ServiceSphere logo displays cleanly without white corners.
- Restored the original ServiceSphere icon style and improved icon rendering to remove white corners.
- Improved launcher icon safe-zone so the ServiceSphere icon appears correctly in the app drawer.
- Manage clients and service jobs.
- Create quotes and invoices.
- Add line items, tax, discounts, and totals.
- Generate and share PDFs.
- Capture photo proof.
- Capture client signatures.
- Schedule jobs and reminders.
- Send follow-up messages.
- Export or delete local data.
- Optional Pro subscription support.
