# GitHub Pages Setup

Use GitHub Pages to host the ServiceSphere privacy policy and terms pages from the `/docs` folder.

## Manual Steps

1. Push repo to GitHub.
2. Go to GitHub repository.
3. Open Settings -> Pages.
4. Set Source to Deploy from a branch.
5. Set Branch to `main`.
6. Set Folder to `/docs`.
7. Save.
8. Wait for Pages deployment.

## URLs

Privacy URL will be:

```text
https://<github-username>.github.io/<repo-name>/privacy.html
```

Terms URL will be:

```text
https://<github-username>.github.io/<repo-name>/terms.html
```
